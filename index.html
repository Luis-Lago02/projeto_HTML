O que é HTML?

O HTML é uma Linguagem de Marcação que estrutura a página web, como se fosse o esqueleto do site. Utilizado para criar seções, parágrafos, links usando elementos, tags e atributos.

Tags e Atributos ensinados no curso

  • Cabeçalho
  • Paragrafo
  • Negrito
  • Link
  • lista ordenada
  • lista não ordenada

Cabeçalho

A tag header é um elemento HTML que representa o cabeçalho de uma página web.

Paragrafo

O elemento HTML A Tag p em HTML representa um parágrafo

Negrito

O elemento HTML Strong dá ao texto uma forte importância, e é tipicamente mostrado em negrito.

Link

A tag HTML a href = "link" define um hiperlink.

Lista ordenada

As tags usadas para criar uma lista ordenada são ol e li. A tag ol é usada para definir a lista e a tag li é usada para cada item da lista

Lista não ordenada

As tags usadas para criar uma lista não ordenada são ul e li. A tag ul é usada para definir a lista e a tag li é usada para cada item da lista
